# Password Strength Analyzer System

A full-stack cybersecurity web application built with Angular, Node.js, Express, and MongoDB that analyzes password strength, detects weak passwords, provides security recommendations, and helps prevent common password-related attacks such as brute-force attacks, dictionary attacks, and credential guessing.

## Features
- Real-time password strength analysis
- Weak / Medium / Strong password detection
- Checks password length
- Detects uppercase letters
- Detects lowercase letters
- Detects numbers
- Detects special characters
- Provides password improvement suggestions
- Stores password analysis history
- User-friendly interface

## Tech Stack
### Frontend
- Angular

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## Cybersecurity Concepts
- Password Security
- Brute Force Attack Prevention
- Dictionary Attack Awareness
- Secure Password Best Practices

## Project Goal
This project helps users create stronger passwords and improve cybersecurity awareness by identifying weak passwords before they can be exploited.
